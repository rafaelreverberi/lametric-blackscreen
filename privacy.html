Privacy Policy – BlackScreen for LaMetric

Effective date: 2025-08-31

Who we are

BlackScreen is a minimal LaMetric app that displays a completely black screen.

What data we process

  • The app does not collect, store, or process any personal data.
  • No usage profiles, cookies, trackers, or analytics tools are used.
  • The app only fetches a static JSON file hosted on GitHub Pages, which defines an empty frame. This file contains no personal information.

Purpose of processing

The sole purpose of this app is to display a black screen on the LaMetric device.
